Image Prompt Generator 是什么？

      

        

          这个网站和 YouMind 那类提示词页面有什么不同？
          
页面结构参考“上方生成、下方案例流”的使用方式，但内容来自本地开源提示词集合和自有 Seedream 模板，并保留作者/来源信息。

        

        

          提示词可以直接复制到 GPT Image 2 吗？
          
可以。每张卡片都有完整提示词、图片预览、作者来源和复制按钮，也可以先在顶部生成框改写再使用。

        

        

          后续能接 Seedream 5 Pro 吗？
          
可以。当前第一版先做静态站和提示词库，下一步可以把生成按钮接到 Cloudflare Worker 或 Vercel API，避免在前端暴露密钥。
